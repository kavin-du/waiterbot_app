# waiterbot_app

## Mobile Application for the Waiterbot System Project

### Supports Cross platforms (Android, iOS)

**_Developed using:_** Flutter

### Features

* Real time notifications
* Guest account available
* Credit/Debit card payments
* QR Code Scan
